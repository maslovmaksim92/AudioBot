# Руководство пользователя - Модуль "Финансы"

## 📊 О модуле

Модуль "Финансы" позволяет вести учёт всех денежных операций компании и автоматически генерировать финансовые отчёты.

**URL:** https://smart-agent-system.preview.emergentagent.com/#/finances

## 🚀 Как добавлять данные

### Способ 1: Ручной ввод (одна транзакция)

1. Откройте раздел **"Финансы"** в боковом меню
2. Нажмите кнопку **"Добавить транзакцию"** (зелёная кнопка справа вверху)
3. Заполните форму:
   - **Тип операции**: Доход или Расход
   - **Дата и время**: Когда произошла операция
   - **Сумма**: Сумма в рублях
   - **Категория**: Выберите из списка (зарплата, материалы, оплата услуг и т.д.)
   - **Контрагент**: От кого получили / Кому заплатили (опционально)
   - **Способ оплаты**: Наличные, банковский перевод, карта и т.д. (опционально)
   - **Проект**: Если транзакция относится к конкретному проекту (опционально)
   - **Описание**: Комментарий к транзакции (опционально)
4. Нажмите **"Добавить"**

### Способ 2: Импорт из CSV (много транзакций)

#### Формат CSV файла:

Создайте Excel или CSV файл со следующими колонками:

```csv
date,amount,category,type,description,payment_method,counterparty
2025-01-10T10:00:00,150000,Оплата услуг,income,Оплата от клиента,Банковский перевод,ООО Комфорт
2025-01-11T14:30:00,85000,Зарплата,expense,Зарплата за январь,Банковский перевод,Иванов И.И.
2025-01-12T09:15:00,25000,Материалы,expense,Моющие средства,Наличные,ООО Химснаб
```

**Обязательные колонки:**
- `date` - дата в формате ISO (2025-01-10T10:00:00)
- `amount` - сумма (число)
- `category` - категория (текст из списка категорий)
- `type` - тип (income или expense)

**Опциональные колонки:**
- `description` - описание
- `payment_method` - способ оплаты
- `counterparty` - контрагент
- `project` - проект

#### Как импортировать:

**Через API (curl):**
```bash
curl -X POST http://localhost:8001/api/finances/transactions/import-csv \
  -F "file=@transactions.csv"
```

**Через интерфейс:**
Кнопка "Импорт CSV" в разработке (появится в следующей версии)

### Способ 3: Через API (для программистов)

Используйте POST запрос:

```bash
curl -X POST http://localhost:8001/api/finances/transactions \
  -H "Content-Type: application/json" \
  -d '{
    "date": "2025-01-10T10:00:00",
    "amount": 150000,
    "category": "Оплата услуг",
    "type": "income",
    "description": "Оплата за уборку",
    "counterparty": "ООО Компания",
    "payment_method": "Банковский перевод"
  }'
```

## 📈 Разделы отчётов

### 1. Движение денег (Cash Flow)
- Показывает поступления и расходы по дням
- График движения денежных средств
- Чистый денежный поток

### 2. Прибыли и убытки (P&L)
- Выручка, расходы и прибыль по месяцам
- Маржа (рентабельность)
- Детальная таблица

### 3. Баланс (Balance Sheet)
- Активы (денежные средства, дебиторка, запасы, недвижимость)
- Обязательства (кредиторка, долги)
- Капитал

### 4. Анализ расходов
- Распределение расходов по категориям
- Круговая диаграмма
- Процентное соотношение

### 5. Задолженности
- Список всех долгов
- Активные и просроченные
- Кредиты, лизинг, кредиторская задолженность

### 6. Товарные запасы
- Остатки на складе
- Стоимость запасов
- По категориям

## 🔧 API Endpoints

### Категории
```
GET /api/finances/categories
```
Получить список категорий доходов и расходов

### Транзакции

**Создать:**
```
POST /api/finances/transactions
```

**Получить список:**
```
GET /api/finances/transactions?type=income&date_from=2025-01-01&limit=100
```

**Получить одну:**
```
GET /api/finances/transactions/{id}
```

**Обновить:**
```
PUT /api/finances/transactions/{id}
```

**Удалить:**
```
DELETE /api/finances/transactions/{id}
```

**Импортировать CSV:**
```
POST /api/finances/transactions/import-csv
```

### Отчёты

## ❓ FAQ

**Q: Как удалить транзакцию?**  
A: Через API: `DELETE /api/finances/transactions/{id}` (UI для удаления в разработке)

**Q: Можно ли редактировать транзакцию?**  
A: Да, через API: `PUT /api/finances/transactions/{id}` (UI для редактирования в разработке)

**Q: Откуда берутся данные в отчётах?**  
A: Сейчас показываются демо-данные. После добавления ваших транзакций, отчёты будут автоматически пересчитываться из БД (в следующей версии)

**Q: Где хранятся данные?**  
A: В PostgreSQL базе данных (Yandex Cloud), таблица `financial_transactions`

**Q: Можно ли экспортировать данные?**  
A: Да, через API можно получить все транзакции в JSON формате
