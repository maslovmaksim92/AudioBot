# ⚡ Быстрый старт: Настройка Webhook Novofon

## 🎯 Что нужно сделать (3 простых шага)

### 1️⃣ Откройте Novofon
Зайдите в личный кабинет: https://my.novofon.com

### 2️⃣ Создайте Webhook
- Раздел: **"Интеграции"** → **"Уведомления о событиях"**
- Нажмите: **"Добавить уведомление"**

### 3️⃣ Заполните форму

```
┌─────────────────────────────────────────────────┐
│ Название:                                       │
│ AudioBot Транскрипция                           │
├─────────────────────────────────────────────────┤
│ Тип события:                                    │
│ ☑ Завершение входящего звонка                   │
│ ☑ Завершение исходящего звонка                  │
├─────────────────────────────────────────────────┤
│ URL:                                            │
│ https://audiobot-qci2.onrender.com/api/        │
│ call-summary/webhook/novofon                    │
├─────────────────────────────────────────────────┤
│ Метод: POST                                     │
├─────────────────────────────────────────────────┤
│ Фильтр (опционально):                           │
│ Номер: +79843330712                             │
└─────────────────────────────────────────────────┘
```

**Нажмите "Сохранить"** ✅

---

## 🧪 Как проверить что работает

1. **Сделайте тестовый звонок** с/на +79843330712
2. **Подождите 1-2 минуты** (транскрипция + AI-анализ)
3. **Проверьте Telegram группу** -5007549435

Должна появиться красивая карточка звонка! 🎉

## ❓ Частые вопросы

**Q: Куда скопировать URL?**  
A: Скопируйте полностью:
```
https://audiobot-qci2.onrender.com/api/call-summary/webhook/novofon
```

**Q: Какой метод выбрать?**  
A: Обязательно **POST**

**Q: Нужны ли токены/ключи?**  
A: Нет! Webhook публичный, просто вставьте URL

**Q: Сколько ждать после звонка?**  
A: 1-2 минуты до появления карточки в Telegram

## 🆘 Если не работает

### Проверьте что backend запущен:
Откройте в браузере:
```
https://audiobot-qci2.onrender.com/api/health
```
Должно быть: `{"status":"ok"}`

### Проверьте URL webhook:
Должен быть точно:
```
https://audiobot-qci2.onrender.com/api/call-summary/webhook/novofon
```
(без пробелов, без лишних символов)

### Проверьте что выбран POST
Метод должен быть **POST**, не GET
