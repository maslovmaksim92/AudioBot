# Инструкция по использованию .bat файлов

## Файлы созданы:

### 1. `connect_and_monitor.bat`
**Что делает:**
- Подключается к серверу
- Автоматически запускает мониторинг INVITE запросов
- Показывает входящие звонки в реальном времени

**Как использовать:**
1. Скопируйте файл на рабочий стол или в удобную папку
2. Дважды кликните по файлу
3. Окно PowerShell откроется автоматически
4. Сделайте звонок из приложения
5. Смотрите логи в окне

**Для выхода:** Нажмите `Ctrl+C`, затем `exit`

---

### 2. `quick_connect.bat`
**Что делает:**
- Просто подключается к серверу
- Вы можете вводить команды вручную

**Как использовать:**
1. Дважды кликните по файлу
2. Вы окажетесь в консоли сервера как пользователь `ubuntu`
3. Введите `sudo su` для получения прав root
4. Выполняйте любые команды

---

## Установка:

### Шаг 1: Скопируйте файлы
Скопируйте файлы `.bat` из папки `/app/asterisk/` на ваш компьютер Windows.

Рекомендуемое место: `C:\Users\Администратор\Desktop\`

### Шаг 2: Проверьте путь к SSH ключу
Если ваш SSH ключ находится в другом месте, откройте `.bat` файл блокнотом и измените путь:

```
C:\sshkeys\yc_ru_2025
```

На свой путь, например:
```
C:\Users\Администратор\.ssh\yc_ru_2025
```

### Шаг 3: Готово!
Теперь просто дважды кликайте по файлу для подключения.

---

## Полезные команды на сервере:

После подключения через `quick_connect.bat`:

```bash
# Стать root
sudo su

# Мониторинг всех INVITE
tail -f /var/log/asterisk/full | grep INVITE

# Мониторинг только от LiveKit
tail -f /var/log/asterisk/full | grep livekit-endpoint

# Проверить статус endpoints
asterisk -rx "pjsip show endpoints"

# Перезагрузить конфигурацию
asterisk -rx "pjsip reload"
```

---

## Устранение проблем:

**Ошибка "Permission denied":**
- Проверьте путь к SSH ключу в `.bat` файле
- Убедитесь, что файл `yc_ru_2025` существует

**Окно сразу закрывается:**
- Добавлена команда `pause` в конце
- Если проблема остается, откройте `.bat` файл блокнотом и проверьте синтаксис
